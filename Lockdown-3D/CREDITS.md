# Lockdown — Credits

All third-party assets used in Lockdown are CC0 / public domain.

## Audio (CC0)

- UI button click — Kenney Interface Sounds (kenney.nl, CC0)
- Footsteps (concrete) — OpenGameArt CC0 footstep pack
- Door unlock / locked — freesound.org (CC0 filter)
- Cable plug-in click — freesound.org (CC0 filter)
- Ambient lab hum — freesound.org (CC0 filter)

## Graphics

The geometry in this build is procedurally generated from Unity primitives
(cube, cylinder) inside `LockdownSetupMenu.cs`. No third-party 3D models are
required. Optional polish models (recommended CC0 sources):

- ambientCG — wall and floor textures (CC0)
- Poly Haven — lab props and HDRIs (CC0)
- Sketchfab CC0 filter — additional props

## Fonts

- Built-in Unity LegacyRuntime.ttf (Liberation Sans, Apache-2.0).
